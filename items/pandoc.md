# Pandoc

## 一句话总结

一个通用文档格式转换器，支持 Markdown、HTML、LaTeX、PDF、DOCX、EPUB 等多种格式互转。

## 它解决什么问题

- 文档格式转换通常分散在多套工具里，维护成本高
- 技术写作、出版和知识整理流程需要一个稳定的命令行转换中枢
- 团队想把 Markdown 作为源格式，再输出到多种目标格式时，缺少统一入口

## 适合谁

- 经常处理技术文档、报告和出版内容的人
- 需要在脚本里批量转换文档格式的开发者
- 希望让写作和发布流程更自动化的团队

## 核心能力

- 多格式互转：支持 Word、RTF、ODT、Jupyter、HTML、Markdown、EPUB、PDF 等
- Markdown 增强：支持数学公式、脚注、表格、引用和扩展语法
- 引用系统：内置 citation 和 bibliography 处理能力
- 模板与过滤器：适合按团队流程深度定制输出

## 典型使用场景

- Markdown 转 PDF、DOCX、HTML 或 EPUB
- HTML / Word / wiki 标记和技术文档之间互转
- 在写作流水线中统一做格式编译和导出

## 为什么值得关注

- 它是文档转换领域最经典也最通用的 CLI 之一
- 很多知识工作流和出版工作流，最终都会落到 Pandoc
- 对 AI agent 来说，它也是非常适合生成稳定命令模板的一类工具
