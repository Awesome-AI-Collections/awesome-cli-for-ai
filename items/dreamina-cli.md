# Dreamina CLI

## 一句话总结

一个面向 AI agent 的创作命令行工具，可用即梦 / Dreamina 账号完成文生图、文生视频、图生图和图生视频任务。

## 它解决什么问题

- 图像和视频生成任务通常停留在网页端，难以批量自动化
- agent 想调用创作能力时，需要异步任务、结果下载和历史查询这类命令行抽象
- 创作者和自动化团队希望把多种生成模式统一进脚本流程

## 适合谁

- 想让 AI agent 调用即梦 / Dreamina 创作能力的开发者
- 需要批量跑图、跑视频和轮询任务结果的人
- 想把生成式内容工作流接入终端自动化的团队

## 核心能力

- 多种生成模式：支持文生图、文生视频、图生图和图生视频
- 账号复用：按提供材料，可直接复用现有 Dreamina 账号能力
- 异步任务流程：支持任务轮询、结果下载和历史记录查询
- Agent 友好：定位上明显面向 AI agent 编排和自动创作

## 典型使用场景

- 用提示词批量生成图片或短视频素材
- 把静态图转成动态视频，接进自动化内容流水线
- 让 agent 在后台发起生成任务、轮询结果并收集产物

## 为什么值得关注

- 它代表的是“生成式创作平台能力被收进 CLI”的方向
- 对多轮自动创作和批处理来说，命令行入口比纯网页更有复用价值
- 在中文创作场景里，这类工具很适合进入 agent workflow

## 类似项目

- [ncm-cli](ncm-cli.md) - 同样是消费级服务的 AI CLI，但面向音乐而不是图像 / 视频创作
- [OpenCLI](opencli.md) - 同样强调把平台能力带进终端，但不是专门的创作生成 CLI
